# Faervell AI NPC v0.7.3

## Исправления

- Fandom загружается через стандартный MediaWiki API пакетами ревизий с полным continuation.
- Контрольные страницы (`Королевство Ивелтин`, главная, хронология и таблицы механик) дополнительно разбираются через `action=parse`, чтобы раскрыть шаблоны, карточки и календарь.
- Одиночные Fandom-источники больше не скачиваются как обычный HTML: они также используют MediaWiki API.
- `/stranger knowledge_status probe_api:true` проверяет API и чтение статьи `Королевство Ивелтин` без изменения базы.
- Поиск отдаёт совпадения по названию статьи раньше общих похожих материалов.
- Проект задания создаётся до служебного согласования и содержит название, описание, место, цель и награду.
- После согласования Странник публикует полные условия задания, а не общую фразу.
- Служебные упоминания не попадают в RP-текст.
- Ответ с латиницей, незакрытой репликой или обрывом автоматически отклоняется и запрашивается у другой модели.
- Ответы с `finish_reason`, отличным от `stop`, считаются незавершёнными.
- Тайм-аут ответа увеличен до 180 секунд, лимит актёрского ответа — до 1000 токенов.
- `nvidia/nemotron-3-ultra-550b-a55b:free` поставлена первой среди предпочтительных актёрских моделей.
- Боевые заявления игрока считаются попытками; Странник отвечает действием и самостоятельно разрешает исход, не принимая одностороннее описание за бесспорный факт.
