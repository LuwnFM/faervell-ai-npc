# v0.5 — тестовое появление, блокировка и RP-категории

- `/stranger appear_now` немедленно отправляет видимый RP-пост появления.
- `/stranger movement_lock` временно закрепляет Странника в одном канале.
- `/stranger event_locations` включает и выключает автоматические появления в ивентах.
- `/stranger locations_sync` автоматически регистрирует доступные каналы разрешённых RP-категорий.
- `/stranger permissions` показывает эффективные права в текущем канале.
- Автоматические переходы фильтруются по RP-категориям и фактическим правам Discord.
- Исправлена сортировка импортов, из-за которой падал Ruff в GitHub Actions.
- Пустая база присутствия больше не назначает первую сцену без видимого появления.
