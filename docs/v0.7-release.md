# Faervell AI-NPC v0.7.0 — объединённый исправляющий релиз

## Основные исправления

- Полный импорт Fandom через стандартный MediaWiki API `/ru/api.php`, постраничный `allpages`, повторные попытки и журнал ошибок каждой страницы.
- Индексация Fandom portable-infobox: правители, география, внутриигровая дата, сезон и другие поля карточек больше не теряются.
- Защита старого индекса: неполный crawl меньше 500 страниц не заменяет рабочую базу.
- Hybrid RAG получил русские стемы заголовков (`Ивелтина` → `ивелтин`) и сильный title boost.
- `/stranger knowledge_status` показывает число страниц, фрагментов, свежесть и ошибки последнего импорта.
- Контекст персонажа отделён от других игроков в том же канале; простое сообщение с полным именем считается представлением.
- RP-каналы синхронизируются вместе с категорией и полным путём локации. Общая RP-категория и Фертейт остаются manual-only, ивенты выключены по умолчанию.
- На каждом запуске Странник принудительно блокируется в канале `1488544832950374481`; очередь перемещений очищается. При включённом startup-lock снять блокировку slash-командой нельзя.
- Локальный запрос задания создаёт валидный `QuestDraft` или явную GM-заявку. Заявка хранится в БД и отправляется в настроенный GM-канал с кнопками одобрения/отказа.
- Под последним сообщением каждого ответа находятся модель, spoiler-подсказка и кнопки оценки. Перегенерация доступна только GM/администраторам, использует другую бесплатную модель и редактирует существующие сообщения.
- OpenRouter работает от живого каталога `/models`: бесплатные модели не ограничены whitelist, применяется только blacklist. Платные модели проходят ценовой фильтр `$0.20/M` на вход и выход.
- Запросы OpenRouter отправляют только поддерживаемые параметры; 400 получает минимальный compatibility retry, полный ответ ошибки сохраняется в `model_calls`.
- Добавлена библиотека сценических действий и запрет повторять одну и ту же бытовую сцену подряд.

## После обновления

1. Выполнить `bash scripts/deploy-production.sh` — он применит `scripts/migrate-v0.7.sh` без изменения секретов.
2. В Discord назначить GM-канал: `/stranger gm_channel`.
3. Запустить `/stranger source_ingest`, затем `/stranger knowledge_status`.
4. Проверить `/stranger locations_sync` и `/stranger startup_lock_status`.
5. Спросить про Королевство Ивелтин, текущую дату и локальное задание.
