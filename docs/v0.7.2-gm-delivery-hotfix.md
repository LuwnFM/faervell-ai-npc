# v0.7.2 — надёжная доставка заявок в служебный канал

Исправления:

- служебные слова ГМ/заявка/одобрение удаляются из RP-контекста и локального fallback;
- после каждого ответа бот делает дополнительный sweep ожидающих заявок по guild/scene/channel;
- `/stranger gm_channel` теперь сразу повторно отправляет очередь и показывает число успехов/ошибок;
- приватные ветки поддерживаются: бот пытается разархивировать и присоединиться перед отправкой;
- ошибки прав и Discord API возвращаются в ответ команды и пишутся в Docker-лог;
- нерелевантная статья вроде «Калькулятор рыночной цены» больше не используется как основание локального квеста;
- если commit квеста не вернул review ID, создаётся отдельная служебная заявка;
- пакет версии поднят до `0.7.2`.
