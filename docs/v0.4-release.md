# Faervell AI-NPC v0.4

Версия 0.4 объединяет реестр персонажей v0.2, присутствие и перемещения v0.3 и исправления первого production-запуска.

## Исправления

- В поставку включены модули `services/characters.py` и `services/presence.py`; контейнер больше не падает с `ModuleNotFoundError`.
- Пустой `DISCORD_GM_ROLE_IDS` корректно разбирается как пустой список.
- Slash-команды синхронизируются с настроенной гильдией при старте, а количество команд пишется в лог.
- Добавлена `/stranger commands_sync` и аварийная текстовая команда `!stranger-sync` для GM.
- Кастомный `on_message` теперь вызывает обработчик текстовых команд.
- CI проверяет компиляцию и импорт production-модулей, поэтому неполный патч больше не пройдёт GitHub Actions.
- Добавлен `scripts/deploy-production.sh`, который всегда переходит в правильный каталог и не допускает запуска Compose из домашней директории.

## Старые ответы после ухода Странника

Discord reply считается продолжением разговора только если исходный пост был отправлен во время текущего появления Странника в этой локации.

- Если Странник уже ушёл, reply на оставшийся старый пост архивируется, но ответа не вызывает.
- Такой reply не планирует возвращение и не создаёт запрос на перемещение.
- Если Странник позднее вернулся, reply на пост из предыдущего появления всё равно игнорируется.
- Осмысленный прямой пинг в другой зарегистрированной локации по-прежнему может поставить её следующей целью маршрута.
